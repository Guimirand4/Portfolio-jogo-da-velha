🕹️ Full Stack Tic Tac Toe
Project to create a Tic Tac Toe game with front-end, back-end, and database using HTML, CSS, JavaScript, Node.js, MySQL, and Docker.

🚀 Features

🎮 Play Tic Tac Toe matches in the browser.

🖥️ Node.js APIs to manage matches.

💾 Store game records in a MySQL database.

🔎 Query saved game history.

⚡ Asynchronous communication between front-end and back-end.

🐳 Full environment using Docker containers.
